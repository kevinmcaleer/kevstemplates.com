# Workshop Planner

Effective workshops bring stakeholders together to achieve specific outcomes through structured collaboration. This guide covers planning, facilitating, and following up on project workshops.

---

## Purpose

Well-planned workshops:
- Achieve specific outcomes efficiently
- Engage stakeholders meaningfully
- Build consensus and alignment
- Generate ideas and solutions
- Make decisions collaboratively
- Build team relationships

---

## Types of Project Workshops

| Workshop Type | Purpose | Typical Duration |
|---------------|---------|------------------|
| **Requirements** | Gather and validate needs | Half to full day |
| **Design** | Explore and agree solutions | Half to full day |
| **Planning** | Develop plans collaboratively | 2-4 hours |
| **Risk** | Identify and assess risks | 2-3 hours |
| **Retrospective** | Review and improve | 1-2 hours |
| **Problem-solving** | Resolve specific issues | 1-3 hours |
| **Kick-off** | Launch project/phase | 2-3 hours |
| **Prioritisation** | Rank and select items | 2-3 hours |

## Facilitation Techniques

### Idea Generation

| Technique | Description | Best For |
|-----------|-------------|----------|
| **Brainstorming** | Free-flow idea generation | Quantity of ideas |
| **Brain-writing** | Silent written ideas | Quieter participants |
| **Round-robin** | Each person shares in turn | Equal participation |
| **Mind-mapping** | Visual idea connections | Complex topics |
| **SCAMPER** | Structured prompts | Innovation |
{:class="table table-single table-narrow"}

### Grouping and Analysis

| Technique | Description | Best For |
|-----------|-------------|----------|
| **Affinity mapping** | Group similar items | Organising ideas |
| **Dot voting** | Quick prioritisation | Large lists |
| **MoSCoW** | Must/Should/Could/Won't | Scope decisions |
| **Impact/Effort matrix** | 2x2 grid plotting | Prioritisation |
| **Five Whys** | Root cause analysis | Problem-solving |
{:class="table table-single table-narrow"}

### Decision Making

| Technique | Description | Best For |
|-----------|-------------|----------|
| **Fist of five** | Confidence voting | Quick consensus |
| **Pros/Cons** | List advantages/disadvantages | Evaluating options |
| **Decision matrix** | Score against criteria | Complex decisions |
| **Consent-based** | "Can you live with this?" | Team decisions |
{:class="table table-single table-narrow"}

## Facilitation Tips

### Opening the Workshop

| Activity | Purpose |
|----------|---------|
| Welcome participants | Make comfortable |
| State purpose and objectives | Set expectations |
| Review agenda | Show the plan |
| Set ground rules | Establish norms |
| Icebreaker | Build rapport |
{:class="table table-single table-narrow"}

### Ground Rules

| Rule | Description |
|------|-------------|
| **One conversation** | No side discussions |
| **Respect all views** | No wrong answers |
| **Time-box** | Stick to schedule |
| **Devices away** | Focus on workshop |
| **Participate** | Everyone contributes |
| **Vegas rule** | Safe space for honesty |
{:class="table table-single table-narrow"}

### Managing Dynamics

| Situation | Response |
|-----------|----------|
| **Dominant participant** | Direct to others, use round-robin |
| **Silent participant** | Invite directly, use written input |
| **Off-topic discussion** | Park in "parking lot" |
| **Conflict** | Acknowledge, focus on interests |
| **Energy drop** | Break, energiser, change activity |
| **Running late** | Adjust, prioritise, defer items |
{:class="table table-single table-narrow"}

## Workshop Checklist

### Planning (2+ weeks before)
- [ ] Purpose and objectives defined?
- [ ] Participants identified and invited?
- [ ] Date, time, location confirmed?
- [ ] Agenda drafted?
- [ ] Facilitator and scribe assigned?

### Preparation (1 week before)
- [ ] Agenda finalised and sent?
- [ ] Pre-reading distributed?
- [ ] Materials prepared?
- [ ] Room/technology booked?
- [ ] Catering arranged (if needed)?

### Day Before
- [ ] Room setup confirmed?
- [ ] Technology tested?
- [ ] Materials ready?
- [ ] Participant reminders sent?

### On the Day
- [ ] Arrive early to setup?
- [ ] Test technology?
- [ ] Welcome participants?
- [ ] Follow agenda?
- [ ] Capture outputs?

### After the Workshop
- [ ] Summary email sent?
- [ ] Full write-up completed?
- [ ] Actions tracked?
- [ ] Follow-up scheduled?
