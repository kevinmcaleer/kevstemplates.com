# Training Delivery

Training delivery executes the training plan, ensuring users are prepared to use the new solution effectively.

---

## Purpose

Effective training delivery ensures:
- Users gain required skills
- Confidence in using the solution
- Smooth transition to new ways of working
- Reduced support burden post go-live
- Benefits realisation through adoption

## Pre-Training Preparation

| Activity | Purpose |
|----------|---------|
| **Confirm logistics** | Venue, equipment, materials |
| **Brief trainers** | Content, audience, objectives |
| **Notify attendees** | Schedule, location, preparation |
| **Test environment** | Ensure system access works |
| **Prepare materials** | Print, distribute, upload |

## Effective Facilitation

### Engagement Techniques

| Technique | When to Use |
|-----------|-------------|
| **Demonstrations** | Showing how to do something |
| **Hands-on practice** | Building skills through doing |
| **Group discussions** | Exploring concepts, concerns |
| **Q&A sessions** | Addressing specific questions |
| **Scenario exercises** | Applying learning to real situations |
| **Knowledge checks** | Verifying understanding |
{:class="table table-single table-narrow"}

### Adult Learning Principles

- Adults need to know **why** they're learning
- Adults bring **experience** to draw on
- Adults are **problem-centred**, not content-centred
- Adults need **relevance** to their role
- Adults are **self-directed** learners

## Managing the Training Environment

| Aspect | Best Practice |
|--------|---------------|
| **Physical setup** | Comfortable, visible, accessible |
| **Technical setup** | Test before, have backup |
| **Timing** | Start on time, regular breaks |
| **Materials** | Ready and accessible |
| **Support** | Technical help available |
{:class="table table-single table-narrow"}

---

## Handling Challenges

| Challenge | Response |
|-----------|----------|
| **Technical issues** | Have backup plan, stay calm |
| **Resistant learners** | Acknowledge concerns, show benefits |
| **Mixed skill levels** | Pair strong with developing |
| **Questions off-topic** | Park for later, stay focused |
| **Running over time** | Prioritise key content |
| **Low engagement** | Increase interaction, ask questions |
{:class="table table-single table-narrow"}

---

## Assessing Learning

| Method | Measures |
|--------|----------|
| **Knowledge checks** | Understanding of concepts |
| **Practical exercises** | Ability to perform tasks |
| **Scenario assessments** | Application to real situations |
| **Feedback forms** | Learner perception |
| **Observation** | Demonstrated competence |
{:class="table table-single table-narrow"}

## Measuring Success

| Metric | What It Measures |
|--------|------------------|
| **Attendance** | Did people attend? |
| **Completion** | Did they finish? |
| **Assessment scores** | Did they learn? |
| **Satisfaction** | Did they find it useful? |
| **Application** | Are they using the skills? |
| **Performance** | Is performance improving? |
